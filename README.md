# 📊 Government Data Automation & Ingestion System

## 🚀 Overview
This project focuses on automating the collection and integration of district-level government data (such as tenders, utilization, and political insights) from multiple public websites that do not provide official APIs.

The system was built to support data-driven features in the Vokal platform by ensuring reliable and structured data availability.

---

## ❗ Problem Statement
Many government websites provide valuable public data, but:
- No official APIs are available
- Data is scattered across multiple sources
- Manual extraction is time-consuming and error-prone

The challenge was to **automate data extraction and integrate it into our platform efficiently and reliably**.

---

## 💡 Solution
Developed a complete automation pipeline to scrape, process, and store data:

- Built a **Python automation script** using **Playwright**
- Navigated dynamic websites and extracted data using HTML structure
- Processed and cleaned raw data for consistency
- Sent structured data to backend services via APIs
- Stored data securely in the database

---

## ⚙️ Key Features
- ✅ Automated web scraping from dynamic government websites  
- ✅ No dependency on external APIs  
- ✅ Robust error handling and retry mechanisms  
- ✅ Logging system for tracking execution  
- ✅ Success & failure reports for monitoring data flow  
- ✅ Scalable and reusable scraping architecture  

---

## 🛠️ Tech Stack
- **Python**
- **Playwright**
- **REST APIs**
- **Django (Backend Integration)**
- **Database (for structured storage)**

---

## 🔄 Workflow
1. Script launches and navigates target websites  
2. Extracts required data using HTML selectors  
3. Cleans and structures the data  
4. Sends data to backend APIs  
5. Stores data in database  
6. Logs success, failure, and errors  

---

## 📈 Impact
- Reduced manual effort significantly  
- Enabled real-time availability of government data  
- Improved data reliability and consistency  
- Supported scalable integration for multiple districts
